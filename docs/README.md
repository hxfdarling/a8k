### 简单上手

```bash
# 安装
yarn global add a8k
# 或者：npm install -g a8k

# 新建一个 项目 文件
k create test

# 开始开发
k dev

# 构建发布文件
k build
```

::: warning 注意
请确保你的 Node.js 版本 >= 8。
:::
